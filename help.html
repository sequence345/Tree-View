Instructions

How to set up Tree View to display your own content.

Table layout

1. Populate table:

  • Column A = Item Name
  • Column B = Indentation*
  • Column C = Text
  • Column D = Image Link

*Negative values un-indent.
*Leave empty to accumulate a series.

Publish to web (Shown here using Google Sheets)

2. Go to: File > Share > Publish to web

3. Go to: Web Page > Tab-separated Values

4. Click: Publish

5. Copy: provided link

Load into the Tree View website

6. Open: Tree View

7. Paste link into the input field

8. Click 'Load'
